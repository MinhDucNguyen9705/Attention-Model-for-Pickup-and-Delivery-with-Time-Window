# INSTALLATION

This repository is built in Python 3.12.10 and tested on Windows 11 environment. Please ensure you have Python 3.8 or higher installed.
Follow these instructions

1. Clone the repository:

```powershell
git clone https://github.com/MinhDucNguyen9705/Attention-Model-for-Pickup-and-Delivery-with-Time-Window.git
cd Attention-Model-for-Pickup-and-Delivery-with-Time-Window
```

2. Make conda environment (optional but recommended):

```powershell
conda create -n attention-pd python=3.10 -y
conda activate attention-pd
```

3. Install dependencies:

```powershell
pip install -r requirements.txt
```

4. Download datasets from Google Drive
- [Dataset Link](https://drive.google.com/drive/folders/1STXERYZsyzHCsiFILbzcRKVsPjtGvRlu?hl=vi)
- Place the downloaded datasets in the `data/` directory.
